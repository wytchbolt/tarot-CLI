# tarot-cli
A simple command line interface tool for tarot card readings.

## Installation
Run `npm install -g tarot-cli`

## Usage
`$ tarot-cli`

Choose from a `Daily card`, `One card`, `Three cards`, `Past/Present/Future`, or `Celtic cross` reading.
